# pdfmailer

A utility website that can convert the content of any given url into a pdf and send that pdf to a desired email address.

## Features
-   Conversion of any URL to PDF
-   Send the newly converted PDF to any email address

## Built With
* JavaScript
* NodeJs
* Express
* Puppeteer
* React
